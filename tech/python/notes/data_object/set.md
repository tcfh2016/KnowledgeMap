

## 五、集合

集合是一系列不重复的元素，类似于字典，但只包含键没有值，它的顺序是不确定的。由于它的使用
场景较少，暂不详述。

集合（set）可以用来把重复项从其他集合（collection）中过滤掉，直接把collection转换为set
然后再转换回来即可：

```
l = [1, 2, 1, 3, 2, 4, 5]
l = list(set(l))
```

集合可以求取交集、并集和差集：

```
(set(['b', 'r', 'u', 'o', 'n']), set(['e', 'o', 'g', 'l']))   # 重复的被删除
>>> x & y         # 交集
set(['o'])
>>> x | y         # 并集
set(['b', 'e', 'g', 'l', 'o', 'n', 'r', 'u'])
>>> x - y         # 差集
set(['r', 'b', 'u', 'n'])
```
