# table

HTML页面里面的表格是使用`<table>`标签来定义的，其中的每行使用`<tr>`标签，表格头使用`<th>`定义每个单元格，默认以加粗、居中显示。其他的各行的单元格使用`<td>`进行定义。

表格展示的时候默认没有边框，如果要使用边框那么需要使用CSS的`border`属性。

## 表格样式

### 表格宽度

通过`<table style="width:100%">`指定展示百分比的相对宽度。
