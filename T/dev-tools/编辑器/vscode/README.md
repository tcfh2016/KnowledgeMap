## 快捷键

代码阅读：

- `ctrl + left`: 定义跳转
- `shift + f12`：搜索reference
- `ctrl + t`：全局符号搜索
- `ctrl + shift + o`: 本文件符号搜索
- `ctrl + p`：搜索文件
- `ctrl + shift + F`: 全局搜索字符串


- `alt + left`：后退
- `alt + right`：前进
